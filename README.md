# 💰 Finance Dashboard

## 📌 Overview

This project is a modern finance dashboard built using React. It helps users track financial activity, analyze spending patterns, and gain insights through interactive visualizations.

The goal of this project is to demonstrate frontend development skills including UI design, component structure, and state management.

---

## ✨ Features

### 📊 Dashboard

* Summary cards (Total Balance, Income, Expenses)
* Time-based visualization (monthly trends)
* Category-based visualization (spending breakdown)

### 📄 Transactions

* View all transactions (date, amount, category, type)
* Search by category
* Filter by income/expense
* Sort by amount or date
* Add/Delete transactions (Admin only)

### 🔐 Role-Based UI

* **Viewer** → Can only view data
* **Admin** → Can add & delete transactions
* Toggle between roles (frontend simulation)

### 💡 Insights

* Highest spending category
* Monthly comparison (increase/decrease)
* Category-wise spending percentage
* Monthly income vs expense chart
* Top 3 spending categories

### 🌙 Optional Enhancements

* Dark mode toggle
* Local storage persistence
* CSV export functionality
* Responsive layout
* Smooth UI interactions

---

## 🛠 Tech Stack

* React (Hooks: useState, useEffect, useMemo)
* Recharts (Charts)
* CSS (Custom styling)
* React Icons

## ⚙️ Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/your-username/finance-dashboard.git
```

2. Navigate into the project:

```bash
cd finance-dashboard
```

3. Install dependencies:

```bash
npm install
```

4. Run the app:

```bash
npm run dev
```
